\doxysubsubsection{\texorpdfstring{structure}{structure}}
{\footnotesize\ttfamily enum \mbox{\hyperlink{class_network_a46436c4c473584e8b6fd303ceedb9806}{Network\+::structure}}}

Enums for the structures that can be added

A\+ND\+: Pulses when its two input Neurons pulse

OR\+: Pulses when either of its input Neurons pulse

X\+OR\+: Pulses when only one of its input Neurons pulse (Inputs must pulse within a certain time of each other)

N\+OT\+: Prevents it\textquotesingle{}s reciever from pulsing when its input \mbox{\hyperlink{class_neuron}{Neuron}} pulses

P\+A\+ND\+: \char`\"{}\+Partial\char`\"{} A\+ND, Can be used along with an A\+ND structure to add more necessary inputs. For example instead of needing two input Neurons to pulse one could add a third

P\+OR\+: \char`\"{}\+Partial\char`\"{} OR, Works the same as P\+A\+ND but with OR

T\+I\+M\+ED\+: Pulses to its recievers with a timed delay allowing a N\+OT structure to interupt it

Recieves a list of 1\textquotesingle{}s and 0\textquotesingle{}s indicating whether or not the input \mbox{\hyperlink{class_neuron}{Neuron}} at each index of the inputs list was activated

Activated Neurons pulse to their recievers which in turn pulse to theirs if their critical Charge is reached

This cycle is continued until an output \mbox{\hyperlink{class_neuron}{Neuron}} pulses, all Neurons stop pulsing, or the max amount of pulses is reached
